See how easy that was? You just entered some python code into a coding environment powered by a version of the Python language. When we run code on our machines, we have to have a ‘kernel’ that understands the language we are using and can translate it so that our computer’s resources can do the thing you’ve specified. In this course, we will use the Python and R languages for the most part.

Installing Python and R can be a bit of a nuisance though, since there are so many of you and so many different computers y’all might have. We will be using, for parts of the course, an online version (with some reduced functionality) that uses your browser’s computing resources.
